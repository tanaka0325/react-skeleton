# react-skeleton
skeleton for minimum react application.

## Usage

```sh
# clone and copy
$ git clone https://github.com/tanaka0325/react-skeleton.git
$ cd react-skeleton
$ git checkout-index -f -a --prefix=/path/to/work/

# start
$ cd /path/to/work
$ yarn install
$ yarn start

# access to http://localhost:8765
```
